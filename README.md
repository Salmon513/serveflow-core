# ServeFlow Core

AI-powered restaurant operations platform.

## Vision

ServeFlow Core helps restaurants automate:
- customer communication
- WhatsApp workflows
- booking management
- operational workflows
- AI-assisted customer support

The platform starts as an automation-focused MVP and evolves into a scalable SaaS system.
